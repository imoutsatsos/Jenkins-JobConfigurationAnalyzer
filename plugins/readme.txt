The following plugins are used  in building this project

Required (highest version tested)
	Groovy v 1.29
	Scriptler v 2.9
	Active Choices v 1.5.2
	Summary Report v 1.15
	Workspace Cleanup v 0.32
	Build Name Setter v 1.6.5

Optional (highest version tested)
	Collapsing Console Sections v1.5.0

Dependencies

Please, note that the plugins above may have other dependencies which are not explicitly described here.
For example, a common dependency is the Script Security plugin which is common on most Jenkins Servers that are executing Groovy scripts. Another one is the OWASP Markup Formatter Plugin which is shipped as part of Jenkins and allows the use of a subset of HTML tags to be used in job and parameter description.
